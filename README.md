# Diablo 4 罗盘全自动刷本工具 (Diablo 4 Compass Farm Bot)

这是一个基于图像识别和 OCR 的暗黑破坏神 4 (Diablo 4) 罗盘副本全自动刷本工具。

## 主要功能

*   **全自动流程**：从城镇开始，自动激活罗盘、传送到副本、进入副本、战斗、拾取、回城，循环往复。
*   **智能状态同步**：启动时自动识别当前游戏状态（城镇、波次中、首领房、宝箱房），实现断点续刷。
*   **精准导航**：利用小地图图标（事件图标、血井图标、首领入口等）进行亚像素级的精准定位。
*   **事件识别与选择**：自动识别波次间的“混沌供品”等事件，并根据优先级列表（优先选择混沌）自动点击。
*   **首领选择逻辑**：根据当前以太数量自动选择收益最高的首领入口（如以太 > 1066 自动选择巴图克）。
*   **精准开箱与拾取**：
    *   使用模板匹配和 OCR 双重校验锁定宝箱位置。
    *   **F 键交互**：使用强制交互键 (F) 代替鼠标左键，避免攻击动作干扰。
    *   **两阶段拾取**：先执行“盲吸”确保效率，再通过 OCR 扫描残余物品进行精准补漏。
    *   **笔刷式吸取**：模拟鼠标在掉落物周围快速扫动并配合高频 F 键，极大提高拾取成功率。

## 技术栈

*   **图像处理**：OpenCV (模板匹配)
*   **文字识别**：EasyOCR (波次识别、以太计数、事件文字、掉落标签)
*   **自动化控制**：PyAutoGUI (鼠标移动、点击)、Keyboard (键盘按键模拟)
*   **逻辑控制**：基于状态机的 Python 脚本

## 文件结构

*   `main.py`: 程序入口。
*   `core/agent.py`: 核心状态机逻辑。
*   `core/vision.py`: 视觉识别系统（OCR、截图、模板匹配）。
*   `core/navigation.py`: 导航与操作执行（移动、技能、拾取）。
*   `config.py`: 全局配置（ROI 区域、资产路径、优先级列表等）。
*   `verify_*.py`: 各个环节的独立测试与视觉化调试脚本。

## 使用说明

1.  确保游戏分辨率为 2560x1440（或根据 `config.py` 调整 ROI）。
2.  安装依赖：`pip install -r requirements.txt`。
3.  运行 `python main.py` 启动。
4.  脚本运行期间请勿手动操作鼠标键盘，如需紧急停止请按 `Ctrl+C` 或移动鼠标至屏幕角落（PyAutoGUI 安全保护）。
